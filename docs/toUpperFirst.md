# toUpperFirst

首字母大写

## 版本

0.1.0

## 使用

### 语法

```js
toUpperFirst(str) -> {string}
```

### 参数

| Name   | Type       | Default | Description             |
|--------|------------|---------|-------------------------|
| str    | string     |         | The value to string    |

### 返回值

Returns the string.

## 示例

```js
_.toUpperFirst('abc');
// => Abc
```
