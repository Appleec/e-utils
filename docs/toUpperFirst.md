# toUpperFirst

Converts the first character of `string` to upper case.

## 版本

0.1.2

## 使用

- 语法

```js
toUpperFirst(value) -> {string}
```

- 参数

| Name     | Type       | Default | Description             |
|----------|------------|---------|-------------------------|
| value    | string     |         | The value to string    |

- 返回值

Returns the string.

## 示例

```js
toUpperFirst('fred')
// => 'Fred'

toUpperFirst('FRED')
// => 'FRED'
```
