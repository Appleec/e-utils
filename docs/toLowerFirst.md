# toLowerFirst

Converts the first character of `string` to lower case.

## 版本

0.1.2

## 使用

- 语法

```js
toLowerFirst(value) -> {string}
```

- 参数

| Name    | Type  | Default   | Description                |
|---------|-------|-----------|----------------------------|
| value   | *     | ''        | The value to convert       |

- 返回值

Returns the converted string.

## 示例

```js
toLowerFirst('Fred')
// => 'fred'

toLowerFirst('FRED')
// => 'fRED'
```
